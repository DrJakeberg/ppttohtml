# Git & Release Workflow

## Initial Commit

```bash
git add .
git commit -m "feat: initial presentation kiosk implementation"
```

## Add Remote

```bash
git remote add origin <your-repo-url>
```

## Push

```bash
git push -u origin main
```

## Tagging

```bash
git tag v1.0.0
git push origin v1.0.0
```
